🔬 Anatomy of a Model Entry

Every model entry has the same 4-5 fields — once you know them, you can read any entry in the catalog.

CODE (server_models.json) 
"kokoro-v1": {
  "checkpoint": "mikkoph/kokoro-onnx",
  "recipe": "kokoro",
  "suggested": true,
  "labels": ["tts", "speech"],
  "size": 0.34
}
PLAIN ENGLISH

"kokoro-v1" — The model name. Type this in the CLI: lemonade run kokoro-v1

"checkpoint" — Download from Hugging Face: user "mikkoph", repo "kokoro-onnx"

"recipe": "kokoro" — Use the Kokoro ONNX inference engine (specialized for text-to-speech)

"suggested": true — Show in the "Recommended" list in the Model Manager UI

"labels" — Tags used to filter models in the UI (tts = text-to-speech)

"size": 0.34 — Only 340 MB! Tiny. Great for laptops with limited storage.

CODE (server_models.json — with NPU) 
"Whisper-Large-v3-Turbo": {
  "checkpoints": {
    "main": "ggerganov/whisper.cpp:ggml-large-v3-turbo.bin",
    "npu_cache": "amd/whisper-large-turbo-onnx-npu:..."
  },
  "recipe": "whispercpp"
}
PLAIN ENGLISH

"checkpoints" (plural!) — Two separate files: one for CPU/GPU, one for NPU

"main" — The base Whisper model in GGUF format — used when no NPU is present

"npu_cache" — AMD's pre-compiled NPU-optimized audio encoder. Downloaded separately, activates automatically on XDNA hardware

"recipe": "whispercpp" — Use whisper.cpp as the inference engine

🔑
The npu_cache Field — Additive, Not Required

Notice "npu_cache" — this is a pre-compiled version of the audio encoder, optimized for AMD XDNA. Lemonade downloads both files and picks automatically based on your hardware.

⬇️ What happens when you run lemonade pull Phi-4-mini-instruct-Hybrid

1
Catalog Lookup
Look up "Phi-4-mini-instruct-Hybrid" in server_models.json to find its checkpoint URL
2
Download from Hugging Face
Connect to Hugging Face and download ONNX model files (~5.1 GB)
3
Cache Locally
Files saved to ~/.lemonade/models/ — subsequent loads are instant, no re-download
4
Ready to Use
Model appears in the UI and in lemonade list — ready to run immediately
